
# Embedded Systems RC Car Controls Project

## About
Embedded systems class project implementing real-time 
motor control on a microcontroller using C.
Utilized ADC input, PWM output, and hardware interrupts 
to control RC car movement and speed.

## Technologies
- C
- ADC (Analog to Digital Conversion)
- PWM (Pulse Width Modulation)
- Hardware Interrupts
- Microcontroller Programming
